Nobel Prize Data Analysis

Exploring trends, patterns, and insights from over a century of Nobel Prize data

Project by jjiyapatell

Analysis Highlights

  • Total Nobel laureates analyzed: 989 individuals and organizations
  • Countries with most prizes: United States, United Kingdom, Germany
  • Gender distribution: 94% male, 6% female laureates
  • Year range: 1901 to 2020
  • Most common category: Physics and Chemistry
  • Youngest laureate: Malala Yousafzai (age 17)
  • Oldest laureate: John B. Goodenough (age 97)

Key Insight

The United States has dominated Nobel Prize wins since World War II, accounting for nearly 40% of all prizes in the Sciences categories.

Data & Methodology

This analysis examines Nobel Prize laureates from 1901 to 2020. The dataset includes information about each laureate's name, category, year, motivation, and share of the prize.

Dataset

The analysis uses the laureates.csv file containing detailed information about each Nobel laureate.

Tools Used

  • Python for data analysis
  • Pandas & NumPy for data manipulation
  • Matplotlib & Seaborn for visualization
  • Jupyter Notebook for interactive analysis
